✦ Experimental CSS Animations

Living, breathing, evolving visual effects

🎨 Living Animations

CSS animations that feel alive, responsive, and unpredictable. These effects create the illusion of life through carefully crafted timing and transitions.

💫 Living Color

/* Organic color transitions */
@keyframes living {
  0% { background: #667eea; }
  25% { background: #764ba2; }
  50% { background: #f093fb; }
  75% { background: #f5576c; }
  100% { background: #667eea; }
}

💡 When to use:

Brand avatars, profile pictures, or any UI element that needs to feel alive and organic. Great for creating warmth in interfaces.

🔷 Morphing Shape

/* Shape transformation */
@keyframes morph {
  0%, 100% {
    border-radius: 50%;
    transform: rotate(0deg);

💡 When to use:

Loaders, transitions between states, or abstract art pieces. Perfect for showing transformation and flow in your design.

🎪 Visual Tricks

Creative CSS techniques that play with perception, light, and motion. These effects demonstrate the creative potential of CSS animations.

👻 Flicker

/* Random flickering effect */
@keyframes flicker {
  0%, 19.999%, 22%, 62.999%, 64%, 64.999%, 70%, 100% { opacity: 1; }
  20%, 21.999%, 63%, 63.999%, 65%, 69.999% { opacity: 0.4; }
}

💡 When to use:

Light bulbs, old signs, horror themes, or creating tension. The unpredictable flicker feels authentic and organic.

🎈 Squish

/* Elastic deformation */
@keyframes squish {
  0%, 100% { border-radius: 50%; transform: scale(1, 1); }
  25% { border-radius: 60% 40% 33% 67% / 55% 45% 55% 45%; transform: scale(1.1, 0.9); }
  50% { border-radius: 33% 67% 55% 45% / 45% 55% 45% 55%; transform: scale(0.9, 1.1); }
}

💡 When to use:

Bouncy UI elements, playful buttons, or anything needing a fun, elastic feel. The squishing motion feels tactile and responsive.

🐍 Snake

/* Clip-path polygon morphing */
@keyframes snake {

  25% { clip-path: polygon(50% 50%, 100% 0%, 0% 100%); }
  50% { clip-path: polygon(50% 50%, 100% 0%, 100% 100%); }
  75% { clip-path: polygon(50% 50%, 100% 0%, 100% 100%, 0% 100%); }
  100% { clip-path: polygon(50% 50%, 0% 0%, 0% 100%, 100% 100%); }
}

💡 When to use:

Snake game themes, abstract art, or polygonal design systems. The clip-path animation creates a mesmerizing geometric effect.

💡 Pro Tips

  • Performance: Use transform and opacity for smooth 60fps animations
  • Timing: Match animation duration to the content's purpose (fast for feedback, slow for ambiance)
  • Accessibility: Respect prefers-reduced-motion for users who need less motion
  • Combination: Layer multiple animations using animation shorthand for complex effects
  • Testing: Check animations on mobile devices where GPU acceleration varies
