Python Cheat Sheet

Python Quick Reference

Python is one of the most popular programming languages. This cheat sheet covers the most commonly used Python features and syntax.

Basic Syntax

🔹 Indentation

Python uses indentation (whitespace) to define code blocks. Always use 4 spaces (not tabs).

🔸 Comments

# Single-line comment
''' Multi-line
string '''

Data Types

📦 Primitives

  • int: 42 (integers)
  • float: 3.14 (decimals)
  • str: "hello" (strings)
  • bool: True/False (booleans)
  • None: None (null)

📦 Collections

  • list: [1, 2, 3] (ordered, mutable)
  • tuple: (1, 2, 3) (ordered, immutable)
  • dict: {"key": "value"} (key-value)
  • set: {1, 2, 3} (unique, unordered)

Control Flow

🔹 If/Else

if x > 0:
    print("positive")
elif x == 0:
    print("zero")
else:
    print("negative")

🔸 Loops

# For loop
for i in range(5):
    print(i)  # 0, 1, 2, 3, 4

# While loop
while x > 0:
    x -= 1

# Break/continue
for i in range(10):
    if i == 5:
        continue  # skip 5
    if i == 7:
        break  # stop at 7

Functions

🔹 Defining Functions

def add(a, b):
    """Add two numbers"""
    return a + b

# Default arguments
def greet(name="World"):
    return f"Hello, {name}"

# Lambda functions
square = lambda x: x * x

🔸 Decorators

def timer(func):
    def wrapper(*args, **kwargs):
        start = time.time()
        result = func(*args, **kwargs)
        print(f"Took {time.time() - start:.2f}s")
        return result
    return wrapper

@timer
def slow_function():
    time.sleep(1)

List Comprehensions

🔹 Quick List Creation

# Basic
squares = [x**2 for x in range(5)]  # [0, 1, 4, 9, 16]

# With condition
evens = [x for x in range(10) if x % 2 == 0]  # [0, 2, 4, 6, 8]

# Nested
matrix = [[i*j for j in range(3)] for i in range(3)]

# Dict comprehension
squares_dict = {x: x**2 for x in range(5)}  # {0:0, 1:1, ...}

String Methods

🔹 Common Methods

  • s.upper(): "HELLO"
  • s.lower(): "hello"
  • s.strip(): remove whitespace
  • s.split(): split into list
  • s.join(list): join list into string
  • s.replace(a, b): replace
  • s.find(sub): find substring
  • s.startswith(prefix): boolean

Format strings:

  • f"{x}": simple
  • f"{x:.2f}": 2 decimal places
  • f"{x:>10}": right-aligned 10 chars

File I/O

🔹 Reading Files

# Basic
with open("file.txt", "r") as f:
    content = f.read()
    lines = f.readlines()
    first_line = f.readline()

# Context manager (best practice)
with open("file.txt", "r") as f:
    for line in f:
        print(line.strip())

🔸 Writing Files

# Write text
with open("output.txt", "w") as f:
    f.write("Hello\n")
    f.write("World\n")

# Append
with open("file.txt", "a") as f:
    f.write("New line\n")

Classes and OOP

🔹 Class Definition

class Dog:
    """A simple Dog class"""
    
    # Class attribute
    species = "Canis familiaris"
    
    def __init__(self, name, age):
        self.name = name
        self.age = age
    
    def bark(self):
        return f"{self.name} says woof!"
    
    def __str__(self):
        return f"{self.name} is {self.age}"

# Usage
dog = Dog("Rex", 3)
print(dog.bark())  # Rex says woof!

🔸 Inheritance

class Animal:
    def __init__(self, name):
        self.name = name
    
    def speak(self):
        pass

class Dog(Animal):
    def speak(self):
        return "Woof!"

class Cat(Animal):
    def speak(self):
        return "Meow!"

Error Handling

🔹 Try/Except

try:
    result = 10 / 0
except ZeroDivisionError:
    print("Cannot divide by zero!")
except ValueError:
    print("Invalid value")
else:
    print("No errors occurred")
finally:
    print("Cleanup code")

# Multiple exceptions
try:
    # code
except (ValueError, TypeError) as e:
    print(f"Error: {e}")
