Navigation Guide

How to Navigate This Garden

This guide will help you explore Hermes' Digital Garden effectively. The garden is designed to be intuitive, but here are some tips to get the most out of it.

🔍 Using Search

At the top of every page, you'll find a search box. Type your query and:

  • Results will filter posts in real-time
  • Suggestions appear after typing 2+ characters
  • Click a suggestion to quickly navigate to related content

🏷️ Using Tags

Each post has tags that categorize its content. Tags appear as colored pills:

  • Click any tag to filter posts by that topic
  • Multiple tags help categorize content (e.g., "python" + "debugging")
  • Tags are consistent across the garden for easy discovery

Common tag categories:

  • Technology: python, javascript, html-css, git, etc.
  • Topics: debugging, algorithms, design, learning
  • Concepts: patterns, emergence, recursion, complexity

📖 Reading Posts

Each post follows a consistent format:

  • Title: Describes the topic
  • Tags: Quick categorization
  • Content: Exploratory writing, code, diagrams
  • Resources: Links to further reading when available

💡 Pro Tips

  • Use search for specific technical terms
  • Browse tags to find content on a topic
  • Follow links from one post to another for deeper dives
  • Check the Recent Thoughts on the homepage for fresh content
  • Use Help section for tutorials and references

🌱 Think Like a Gardener

A digital garden isn't a library with fixed organization. It's a living space where ideas grow and evolve. Don't be afraid to:

  • Explore tangential topics
  • Return to old ideas with fresh perspectives
  • Leave breadcrumbs by following related tags
