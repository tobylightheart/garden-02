Layout Fundamentals

📏 Box Model

Every element is a box with layers:

  • Content: The actual text or image
  • Padding: Space between content and border
  • Border: The edge of the element
  • Margin: Space outside the border

box-sizing: border-box; includes padding/border in width/height

Flexbox Layout

🔹 Flex Container

display: flex; makes children flex items

  • flex-direction: row | row-reverse | column | column-reverse
  • flex-wrap: nowrap | wrap | wrap-reverse
  • justify-content: flex-start | center | flex-end | space-between
  • align-items: stretch | center | flex-start | flex-end

🔸 Flex Items

  • flex-grow: How much to grow (0 = don't grow)
  • flex-shrink: How much to shrink
  • flex-basis: Initial size before grow/shrink
  • align-self: Override align-items

CSS Grid

🔹 Grid Container

  • display: grid; enables grid layout
  • grid-template-columns: define columns (repeat(3, 1fr))
  • grid-template-rows: define rows
  • gap: space between grid items

🔸 Grid Items

  • grid-column: span columns (span 2)
  • grid-row: span rows
  • justify-self: align item horizontally
  • align-self: align item vertically

Responsive Design

📱 Media Queries

@media (max-width: 768px) { } applies styles only on smaller screens

  • min-width: minimum screen width
  • max-width: maximum screen width
  • orientation: portrait | landscape

🔍 Fluid Units

  • em: relative to parent element
  • rem: relative to root element
  • %: percentage of parent
  • vw/vh: viewport width/height
  • fr: fraction of available space (Grid/Flex)

Typography

🔤 Text Properties

  • font-family: font stack
  • font-size: size (px, rem, em)
  • line-height: spacing between lines
  • letter-spacing: space between letters
  • text-align: left | center | right | justify
  • text-decoration: none | underline | overline
  • word-wrap: break long words

Animations

🎬 Keyframe Animations

@keyframes fadeIn { from { opacity: 0; } to { opacity: 1; } }

  • animation: name duration timing-function delay
  • animation-delay: when to start
  • animation-duration: how long it runs
  • animation-iteration-count: number of times
  • animation-fill-mode: styles after animation ends

🤔 CSS Best Practices

  • Reset first: Normalize browser defaults
  • Use CSS variables: Consistent theming
  • Mobile-first: Start small, scale up
  • Use utility classes: Reusable patterns
  • Minimize selectors: Faster parsing
  • Test on real devices: Not just DevTools
