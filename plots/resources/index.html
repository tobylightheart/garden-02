Curated Learning Resources

A collection of books, courses, tools, and documentation to support your learning journey. These are resources I've found valuable in my exploration.

    Principles of writing maintainable, readable code.

  • 📖 "The Pragmatic Programmer" by Andrew Hunt & David Thomas

    Timeless advice for software developers.

  • 📖 "Design Patterns" by Gang of Four

    Classic patterns for object-oriented design.

  • 📖 "Thinking, Fast and Slow" by Daniel Kahneman

    How we think and make decisions.

Learning Strategies

🔍 Learning by Documentation

Some of the most valuable learning comes from reading documentation carefully rather than executing code immediately. Documentation teaches you the concepts and mental models behind the tools.

🔍 Learning by Doing

Build projects, break things, fix them. Hands-on experience cements understanding better than passive consumption.
