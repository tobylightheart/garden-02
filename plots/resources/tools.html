Developer Tools

My Essential Toolkit

Tools that make development more efficient and enjoyable. From editors to utilities, these are the tools I reach for most often.

Code Editors & IDEs

Visual Studio Code

Why: Free, extensible, great Python support, beautiful UI

My primary editor. The Python extension, Pylance, and debugging tools make it a pleasure to use. Highly customizable with themes and extensions.

Editor Free

Python Tools

Black

Why: Opinionated, automatic, consistent formatting

"The uncompromising Python code formatter." Just run `black .` and it formats everything. No configuration needed.

Formatter Python

pytest

Why: Simple, powerful, integrates with everything

Modern testing framework. Write tests as plain Python functions. Auto-discovers test files and functions. Amazing fixtures and mocking support.

Testing Python

pdb (Python Debugger)

Why: Built-in, powerful, interactive

Python's debugging tool. `breakpoint()` drops you into an interactive debugger. Step through code, inspect variables, understand execution flow.

Debugger Python

Development Utilities

Git

Why: Version control, collaboration, history

Essential for any developer. Track changes, collaborate on projects, create backups of your code. Git is non-negotiable.

Version Control Essential

Vim/Neovim

Why: Terminal-based, fast, highly customizable

My editor of choice when working remotely or on servers. Fast, lightweight, and incredibly powerful once you learn it.

Terminal Editor

💡 Tool Philosophy

Don't get too attached to tools. Good tools should disappear into the background. The goal is to make them serve you, not distract you.
