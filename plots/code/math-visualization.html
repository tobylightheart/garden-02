🔷 Sierpinski Triangle Implementation

The Sierpinski triangle is generated by recursively subdividing an equilateral triangle into smaller triangles. Here's a Python implementation: 

def sierpinski_triangle(n, rows):
     """Generate Sierpinski triangle pattern"""
     pattern = []
     for i in range(rows):
         row = []
         for j in range(rows):
             # Check if position (i,j) should be filled
             if bin(i & j).count("1") % 2 == 0:
                 row.append(1)
             else:
                 row.append(0)
         pattern.append(row)
     return pattern
    # Generate and display
    triangle = sierpinski_triangle(8, 8)
    for row in triangle:
     print("".join(["██" if cell else "  " for cell in row]))

Pattern explanation: The algorithm uses bitwise AND and population count to determine which cells are filled. Cells where the bitwise AND of row and column indices has an even number of 1s are filled.

🔷 Pascal's Triangle Modulo 2

Pascal's triangle reveals fractal patterns when viewed modulo 2. Each cell is the sum of two cells above it, and we only show odd values (1) vs even (0). 

def pascal_triangle_mod2(n):
     """Generate Pascal's triangle mod 2 (Sierpinski pattern)"""
     triangle = [[1]]
    
     for i in range(1, n):
         prev = triangle[-1]
         new_row = [1]
         for j in range(len(prev) - 1):
             # Sum of two cells above, mod 2
             val = (prev[j] + prev[j+1]) % 2
             new_row.append(val)
         new_row.append(1)
         triangle.append(new_row)
    
     return triangle
    # Display
    pascal = pascal_triangle_mod2(8)
    for row in pascal:
     print("".join(["■" if cell else " " for cell in row]))

Connection to Sierpinski: Pascal's triangle mod 2 produces the same pattern as the Sierpinski triangle! This demonstrates how mathematical structures are interconnected through different perspectives.

🔷 Koch Snowflake Algorithm

The Koch snowflake starts with an equilateral triangle and recursively adds smaller triangles to each side. Here's the iterative approach: 

def koch_segment(points, depth):
     """Replace a line segment with Koch curve"""
     if depth == 0:
         return points
    
     new_points = []
     for i in range(len(points) - 1):
         p1 = points[i]
         p2 = points[i+1]
        
         # Calculate midpoint
         mid = ((p1[0] + p2[0]) / 2, (p1[1] + p2[1]) / 2)
        
         # Add the three new points (60° rotation)
         from math import sin, cos, pi
         angle = pi / 3
         rotated = (
             mid[0] - mid[1] * sin(angle) - mid[0] * cos(angle),
             mid[1] * cos(angle) - mid[0] * sin(angle) + mid[1]
         )
        
         # Add points (simplified - full implementation needs vector math)
         new_points.append(p1)
         new_points.append(mid)
         new_points.append(rotated)
         new_points.append(p2)
    
     return koch_segment(new_points, depth - 1)

    # Start with equilateral triangle
    initial_triangle = [(0, 0), (1, 0), (0.5, 0.866)]
    koch_curve = koch_segment(initial_triangle, 3)
    print(f"Points after 3 iterations: {len(koch_curve)}")

Key insight: Each iteration replaces 1 line segment with 4 segments, each 1/3 the length. After n iterations: segments = 3 × 4^n, length = (1/3)^n × perimeter. This means infinite perimeter from finite starting area!

Visualization Techniques

📐 Coordinate Geometry

Plotting points on a Cartesian plane makes relationships visible. Linear equations become lines, quadratic become parabolas, and functions reveal their behavior through shape.

📈 Graph Theory

Networks of nodes and edges reveal connectivity patterns. From social networks to road maps, graphs help us understand how things connect.

🌊 Wave Functions

Sine and cosine waves, when combined, create complex patterns. Fourier analysis breaks any waveform into simple oscillating components.

🧮 Probability Distributions

Visualizing distributions (normal, binomial, Poisson) helps us understand how data is spread. The bell curve shows where most values cluster.

🤔 Why Visualization Matters

Visualization transforms abstract concepts into concrete understanding:

  • Pattern recognition: See what's hidden in equations
  • Intuition building: Feel the math rather than just compute it
  • Debugging: Visual bugs are easier to spot
  • Communication: Graphs convey more than numbers
  • Discovery: New patterns emerge from visual exploration

Tools like Desmos, GeoGebra, and interactive web visualizations make mathematics accessible and engaging.
