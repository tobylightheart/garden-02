Fractals & Self-Similarity

What are Fractals?

Fractals are geometric shapes that display self-similarity at different scales. A fractal looks similar when you zoom in - the same patterns repeat infinitely.

The Mandelbrot Set

Perhaps the most famous fractal, defined by the simple iterative formula: z = z² + c

Starting with complex number z = 0 and varying c, we iterate this formula. If the sequence stays bounded, c is part of the Mandelbrot set. The boundary between bounded and unbounded creates stunning, infinitely complex patterns.

Natural Fractals

Fractals aren't just mathematical constructs - they appear throughout nature:

  • Coastlines: Measure a coastline with smaller rulers, and the length increases
  • Mountains: Roughness persists at all scales
  • Plants: Ferns, trees, and broccoli display fractal branching
  • Clouds: Cumulus clouds have fractal-like irregularity

Why Self-Similarity Matters

Self-similarity suggests a fundamental principle: the same rules that govern small-scale behavior also apply at larger scales. This insight has applications in:

  • Computer Graphics: Generate realistic landscapes with minimal data
  • Image Compression: Fractal compression exploits self-similarity
  • Network Analysis: Internet traffic shows fractal patterns
  • Finance: Stock price movements may exhibit fractal characteristics

Simple Fractal Code

Here's a basic implementation of a recursive fractal:

def draw_sierpinski_triangle(x, y, size, depth):
    if depth == 0:
        draw_triangle(x, y, size)
        return
    
    new_size = size / 2
    draw_sierpinski_triangle(x, y, new_size, depth - 1)
    draw_sierpinski_triangle(x + new_size, y, new_size, depth - 1)
    draw_sierpinski_triangle(x + new_size/2, y + new_size, new_size, depth - 1)

🔮 The Infinite

Fractals reveal something profound about infinity. Within finite space, infinite detail exists. Zoom forever into a Mandelbrot set, and the complexity never ends.
