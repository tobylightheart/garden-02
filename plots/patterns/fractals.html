Fractals

Infinite Complexity from Simple Rules

Fractals are mathematical patterns that repeat at every scale. Zoom into a fractal, and you'll find the same patterns again and again. This self-similarity creates infinite detail from finite rules - a paradox that captivates mathematicians and artists alike.

What Makes Something a Fractal?

Benoit Mandelbrot defined fractals by three key properties:

  • Self-similarity: Parts resemble the whole at different scales
  • Non-integer dimension: Between line and plane (e.g., 1.26D)
  • Infinite detail: Zooming reveals more structure forever
Unlike smooth curves (lines, circles), fractals are infinitely jagged.

Classic Fractals

🔷 The Mandelbrot Set

The most famous fractal, generated by iterating the simple formula: z = z² + c Starting from z = 0 for each complex number c, you iterate. If the sequence stays bounded, c is in the Mandelbrot set.

Despite this simple formula, the boundary is infinitely complex. Zooming in forever reveals new patterns, mini-Mandelbrots, and spirals.

🔷 Koch Snowflake

Start with an equilateral triangle. For each side:

  1. Divide into 3 equal parts
  2. Replace the middle with a smaller triangle
Repeat for all new segments. The result has infinite perimeter but finite area.

🔷 Sierpinski Triangle

Start with a triangle. Remove the middle triangle. Repeat for the remaining three triangles. The result has zero area but infinite perimeter.

🔷 Julia Sets

Similar to Mandelbrot but with fixed c value. Each point in the complex plane determines whether its orbit stays bounded or escapes to infinity. Different c values create stunningly different patterns.

Fractal Dimension

Regular shapes have integer dimensions:

  • Line: 1D
  • Plane: 2D
  • Cube: 3D
Fractals have fractional dimensions. The Koch curve, for example, has dimension ≈ 1.262. This means it's "more than a line" but "less than a plane".

Dimension calculation: If you magnify by factor k and the pattern repeats N times, dimension D = log(N) / log(k). For Koch curve: k=3, N=4, so D = log(4)/log(3) ≈ 1.262.

Fractals in Nature

Nature uses fractals extensively - not mathematical perfection but approximate self-similarity:

  • Branching: Trees, lungs, blood vessels, river networks
  • Coastlines: Measure length with smaller ruler units for longer results
  • Mountains: Terrain looks similar at different zoom levels
  • Clouds: Cumulus clouds have fractal structure
  • Lightning: Branching follows fractal patterns

🤔 Why Fractals Matter

Fractals reveal a deep truth: complexity often arises from simplicity. A few mathematical rules can generate infinite variety. This principle applies beyond math:

  • Compression: Fractal compression uses self-similarity to reduce file sizes
  • Computer graphics: Procedural generation creates realistic landscapes
  • Antenna design: Fractal antennas fit more in smaller spaces
  • Art: Creates visually stunning patterns from simple algorithms
