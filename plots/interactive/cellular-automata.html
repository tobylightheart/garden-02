Conway's Game of Life

Conway's Game of Life is a cellular automaton that demonstrates how simple rules can produce complex, emergent behavior. Each cell follows basic rules based on its neighbors.

Generation: 0 | Alive cells: 0

How It Works

Each cell is either alive (white) or dead (black). Every generation, cells update based on these rules:

  • Underpopulation: A live cell with fewer than 2 live neighbors dies
  • Survival: A live cell with 2 or 3 live neighbors lives
  • Overpopulation: A live cell with more than 3 live neighbors dies
  • Reproduction: A dead cell with exactly 3 live neighbors becomes alive

Interesting Patterns

  • Glider: A pattern that moves diagonally across the grid
  • Still Life: A pattern that never changes
  • Oscillator: A pattern that repeats after a certain number of generations
  • Methuselah: A pattern that takes many generations to stabilize

🌀 Emergence in Action

The Game of Life shows how simple local rules can create complex global behavior. No cell "knows" it's part of a glider or oscillator - these structures emerge from the interactions.
