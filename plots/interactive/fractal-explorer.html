🤔 Understanding Fractals

Fractals are generated by iterating simple mathematical formulas. The Mandelbrot set uses z = z² + c, where each point in the complex plane determines whether its orbit stays bounded or escapes to infinity.

The color you see represents how quickly points escape - faster escape equals lighter colors. Points that never escape form the black set.

Julia Sets use the same iteration formula but with a fixed complex number c (controlled by the Julia Real/Imag sliders). The Mandelbrot set tells you which Julia sets are connected.
